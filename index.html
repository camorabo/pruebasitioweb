Definición ¿QUÉ ES HTML?

HTML no es un lenguaje de programación, es un lenguaje de etiquetas que permite estructurar el contenido. Las etiquetas de encierre pueden introducir una imagen, cambiar los estilos de la letra o incluso transportarte a otro sitio.

¿No recuerdas como escribir en HTML? Recuerda... Anatomía de un elemento HTML

  1. La etiqueta de apertura: consiste en el nombre del elemento (en este caso, p), encerrado por paréntesis angulares (< >) de apertura y cierre. Establece dónde comienza o empieza a tener efecto el elemento —en este caso, dónde es el comienzo del párrafo—.
  2. La etiqueta de cierre: es igual que la etiqueta de apertura, excepto que incluye una barra de cierre (/) antes del nombre de la etiqueta. Establece dónde termina el elemento —en este caso dónde termina el párrafo—.
  3. El contenido: este es el contenido del elemento, que en este caso es sólo texto.
  4. El elemento: la etiqueta de apertura, más la etiqueta de cierre, más el contenido equivale al elemento.

Las etiquetas pueden tener atributos y estos deben tener siempre:

  1. Un espacio entre el atributo y el nombre del elemento o de atributos previos.
  2. Nombre del atributo junto con el signo de igual (=).
  3. Comillas de apertura y de cierre para encerrar el valor del atributo.

Otra propiedad de HTML es... Anidar elementos

Se pueden incluir elementos dentro de otros elementos, para hacer énfasis y que la palabra quede en negrilla se utiliza la etiqueta p con sus llaves respectivas y dentro se incluye la etiqueta strong encerrando la palabra que se quiere que esté en negrilla.


El orden es importante por lo que es indispensable tener presente que etiquetas tienen que irse cerrando de tal forma que se sepa con claridad que elementos quedan por fuera y cuáles no.

Que no se nos olvide que algunos son... Elementos vacíos

Algunos elementos no poseen contenido, como por ejemplo la etiqueta img, que no tiene etiqueta de cierre y tiene dos atributos: src que nos determina de dónde proviene el recurso y alt, el texto que nos dice de qué trata la imagen cuando se utiliza un lector de pantalla.
