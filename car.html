Características de un documento HTML

Las etiquetas en un documento HTML, no son útiles por sí solas tienen que tener una estructura y existe una jerarquía de etiquetas presente en todo el documento HTML. Desconocer este aspecto de las etiquetas puede dañar por completo el diseño de la página web. 

<!DOCTYPE html>
                                <html>
                                  <head>
                                    <meta charset="utf-8">
                                    <title>Mi pagina de prueba</title>
                                  </head>
                                  <body>
                                    <img src="images/firefox-icon.png" alt="Mi imagen de prueba">
                                  </body>
                                </html>

Se tienen:

  • <!DOCTYPE html> Nos indica que es un tipo de documento HTML, es por otra parte una etiqueta que no tiene cierre.
  • <html></html> — el elemento <html>. Esta etiqueta encierra todo lo que debe contener el documento, tiene dos partes claramente diferenciadas el head y el body es conocido también como el elemento raíz (root).
  • <head></head> — el elemento <head>. Es una especie de contenedor de todo aquello que se incluye en la página y no puede ser visto por los visitantes de la misma. Incluye palabras claves, estilos, etc.
  • <meta charset="utf-8"> — <meta>. Especifica que caracteres se pueden utilizar en este documento.
  • <title></title> — el elemento <title>establece el título la página web, el cual aparece en la pestaña o en la barra de título del navegador, y se usa para describir la página cuando es añadida a los marcadores o como favorita.
  • <body></body> — el elemento <body>. Encierra todo el contenido que deseamos mostrar a los usuarios web que visiten la página, ya sea texto, imágenes, videos, juegos, pistas de audio reproducibles, y demás.

¿Y cómo introducimos Imágenes? IMÁGENES

Las imágenes tienen dos atributos: src y alt, en los cuales el primero src contiene una ruta o ubicación de la imagen y el segundo alt un texto alternativo que describe la imagen para personas no videntes. En otros casos el atributo alt cuando se produce un error y no se carga la imagen, presenta este texto en vez de la imagen.

<img src="images/firefox-icon.png" alt="Mi imagen de prueba">

¿Y los títulos? ENCABEZADOS

Los encabezados de manera análoga a lo que sucede en word, nos permiten dar jerarquía a diferentes títulos para dar categorías a los diferentes tipos de textos que tenemos en una página web. HTML posee hasta seis niveles de encabezados. 

<h1>Mi título principal</h1>
<h2>Mi título de nivel superior</h2>
<h3>Mi subtítulo</h3>
<h4>Mi sub-subtítulo</h4>

¿Y las listas? LISTAS

Las listas son muy importantes ya que junto con los párrafos y los encabezados, son uno de los elementos más usados en las páginas web. Son definidas mediante las etiquetas "ol" cuando están ordenadas y "ul" cuando están desordenadas se encierran con la etiqueta "li" en cada punto de la lista. 

<p>En Mozilla, somos una comunidad de</p>

                        <ul>
                          <li>tecnólogos</li>
                          <li>pensadores</li>
                          <li>constructores</li>
                        </ul>
                        
                        <p>trabajando juntos... </p>
